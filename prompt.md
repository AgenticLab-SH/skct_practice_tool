5. 📝 메모장/그림판
모드 전환: 메모장 ↔ 그림판 토글

메모장: 텍스트 입력 및 편집
그림판: 마우스로 자유롭게 그림 그리기
키보드 이벤트 차단: 사용 시 계산기 키보드 입력 차단}
참고사항{괄호 미지원: 복잡한 식은 (A+B)×C 대신 A+B를 먼저 계산 후 결과값에 ×C를 수행하세요.
메모장 활용: 계산기의 결과값을 Ctrl+C, Ctrl+V로 메모장에 옮겨두고 활용하는 것이 필수적입니다.}
계산기는 일반 계산기(사칙연산 포함)이고, 깔끔한 기본 디자인에 키보드 입력이 가능해.
매우 깔끔하고 간단한 ui가 만들어져야해.
타이머도 매우 깔끔하게 만들어져야해. 예시보다 더 깔끔하고 여백이 없도록. 모든 공간에는 여백이 없는 수준이어야해. 만들어봐.
크기{실 SKCT의 메모장/그림판 크기는 16인치 모니터 기준, 가로세로 7cm 정도이다. 직접 사전점검때 측정하고 이를 기반으로 그림판 크기를 더 줄여 연습했다. 현재 내 모니터는 14인치이다.}
추가정보{실 SKCT는 문제를 넘길때마다 메모장, 그림판, 계산기 모두 초기화된다. 따라서 각 과목에서 내가 몇개를 풀었는지 적어놓을 수 없다

​

하지만 사전점검때 Ctrl+C, Ctrl+V가 되는 것을 확인했고, 이를 기반으로 풀이 개수를 계속 복사하고 갱신하며 기록했다

​

언어추리 풀고 "13" 을 Ctrl+C로 저장하고, 자료해석 풀고 Ctrl+V 로 불러와 "13/15" 적고 다시 저장하는 방식으로 말이다
그리고 사전 점검시 확인한 그림판의 글씨체가 흐물흐물해서, 이를 최대한 모방하고자 했다. 그림판 어플에서 크레용을 택하고, 두께 4px하면 비슷하다. 보다시피 생각보다 글씨가 맹맹한 느낌이다. 이렇게 연습했다.
​

그리고 실SKCT에서는 그림판 어플과 다르게 지우개 버튼을 클릭하면 모든 내역이 지워진다. 따라서 지우개로 조금씩 수정하면서 푸는 방법은 지양하도록 하자}
