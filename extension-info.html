외부성 테스트 자료 안내

이 페이지는 SKCT Tool의 핵심 연습 기능과 분리된 별도 안내 페이지입니다. 아래 ZIP은 브라우저 환경에서만 시험적으로 사용하는 부가 자료이며, 호환성이나 동작 결과를 운영자가 보증하지 않습니다.

무엇인가요?

특정 CBT 사이트의 화면 차단 동작을 브라우저에서 시험적으로 우회하는 별도 크롬 확장 ZIP입니다. 메인 연습 툴의 필수 기능은 아닙니다.

무엇이 다른가요?

메인 사이트는 연습 도구 제공이 목적이고, 이 자료는 부가 테스트 자료입니다. 설치 여부와 사용 판단은 사용자 본인이 직접 결정해야 합니다.

설치 요약

  • ZIP 파일을 내려받습니다.
  • 크롬 `chrome://extensions`에서 개발자 모드를 켭니다.
  • ZIP 또는 압축 해제 폴더를 브라우저에 불러옵니다.
사용 전 꼭 확인하세요 실제 시험 환경, 회사 보안 정책, 외부 서비스 약관, 브라우저 정책과 충돌할 수 있습니다. 이 자료는 연습용 보조 테스트 자료이며, 설치/사용/충돌/차단/오작동에 대한 최종 책임은 사용자에게 있습니다.
ZIP 다운로드 메인 연습 도구로 돌아가기

권장 순서는 먼저 메인 연습 도구만으로 필요한 기능을 확인하고, 별도 테스트가 정말 필요한 경우에만 이 페이지의 자료를 판단해서 사용하는 것입니다.
