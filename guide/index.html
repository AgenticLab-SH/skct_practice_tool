가장 빨리 익혀야 하는 흐름

SKCT Tool은 한 화면에서 OMR, 타이머, 메모장, 그림판, 계산기를 같이 쓰는 연습 도구입니다. 아래 순서대로 보면 헤매지 않고 바로 연습을 시작할 수 있습니다.

연습 도구 열기 자주 묻는 질문 보기

핵심 흐름 5단계

1. 시작

타이머와 연습 모드를 먼저 확인

좌측 설정에서 전체 시간, 과목 시간, 쉬는 시간을 확인한 뒤 타이머를 시작합니다. 기본값은 실제 연습 리듬을 맞추는 기준선으로 유지하는 편이 좋습니다.

2. 풀이

OMR과 보조 도구를 같이 사용

문제를 풀면서 OMR에 답을 체크하고, 메모장이나 그림판은 풀이 보조용으로 씁니다. 계산기는 실제 시험처럼 제한된 조작 흐름을 유지합니다.

3. 쉬는 시간

과목 전환 리듬 익히기

쉬는 시간이 자동으로 들어가면 다음 과목 시작 전까지 흐름을 정리할 수 있습니다. 필요하면 쉬는 시간 건너뛰기를 사용할 수 있습니다.

4. 채점

정답 입력 후 결과 확인

정답 입력 모드로 전환한 뒤 정답을 넣고 채점합니다. 결과 패널에서 맞은 수, 푼 수, 정답률, 건너뛴 수를 한 번에 봅니다.

5. 복기

상세 통계로 약점 정리

과목별 상세 통계와 문항별 소요 시간을 보면 어떤 과목과 어떤 문제에서 막혔는지 빠르게 잡을 수 있습니다.

기능별 빠른 설명

OMR

  • 정답 마킹과 채점을 위한 연습용 OMR입니다.
  • 문항 이동 흐름이 빠르도록 구성되어 있습니다.
  • 건너뛰기와 과목 단위 흐름 보정이 들어 있습니다.

타이머

  • 전체 시간과 현재 과목 시간을 동시에 보여줍니다.
  • 쉬는 시간과 다음 과목 전환이 자동으로 이어집니다.
  • 고급 모드에서는 과목 단위 제어가 더 열립니다.

메모장 / 그림판 / 계산기

  • 풀이 보조용으로 한 화면 안에 같이 붙어 있습니다.
  • 메모장은 텍스트 기록, 그림판은 빠른 도식화에 적합합니다.
  • 계산기는 실전형 제한 흐름을 유지합니다.
일반 가이드는 기본 연습 흐름 설명에 집중합니다. 고급 모드, 자료 보관함, 별도 테스트 자료는 핵심 연습 도구와 역할이 다르므로 메인 연습 흐름과 분리해서 보는 편이 맞습니다.
