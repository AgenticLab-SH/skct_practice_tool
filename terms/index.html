핵심은 세 가지입니다. 무료 기본 도구와 별도 고급 기능은 다르고, 별도 테스트 자료는 메인 앱과 책임 범위가 다르며, 운영자는 서비스 구조를 바꿀 수 있습니다.

무료 / 유료 범위

기본 연습 도구는 무료로 제공됩니다. 고급 기능은 별도 신청, 승인, 계정 확인이 필요한 추가 기능입니다.

금지 행위

  • 서비스 동작을 고의로 방해하는 행위
  • 커뮤니티 스팸, 비방, 불법 콘텐츠 작성
  • 운영자나 다른 사용자의 계정/데이터를 무단으로 다루는 행위
  • 별도 테스트 자료를 메인 앱의 공식 기능처럼 오인하게 퍼뜨리는 행위

서비스 변경 가능성

기능, 문구, 신청 흐름, 자료 보관함 구조는 운영 판단에 따라 변경되거나 중단될 수 있습니다. 특히 고급 기능과 별도 테스트 자료는 구조 조정 가능성이 높습니다.

책임 제한

핵심 연습 도구는 연습 보조 목적입니다. 별도 테스트 자료, 외부 사이트 호환성, 브라우저 확장 충돌, 실전 시험 환경 적합성은 사용자가 직접 판단해야 합니다.

실제 운영 반영 전에는 이 문서를 운영자 기준으로 한 번 더 다듬는 것이 좋습니다. 지금은 코드와 현 운영 구조를 설명하기 위한 로컬 초안 성격이 더 강합니다.
