# 개인 학습자료 저장 뷰어 기능
작성일시: 2026-04-09 16:10:35 KST

개인 학습자료 저장과 뷰어 기능은 이번 라운드에서 로컬 코드와 문서 기준으로 완료했습니다.

## 2026-04-09 처리 결과
- 메인 연습 화면과 섞지 않고 `study-archive.html` 전용 경로로 분리했습니다.
- Firebase Auth Email/Password 기반 로그인과 세션 유지 흐름을 붙였습니다.
- RTDB 저장 경로를 `userStudyLibrary/<uid>/items/<itemId>`로 고정하고, 사용자별 읽기와 쓰기 rules를 추가했습니다.
- 입력 폼에는 제목, 주관사, 회차, 과목, 저장 유형, 문제 형식, 태그, 원문, AI 응답을 넣었습니다.
- 붙여넣은 원문에서 회차, 난이도, 문제 수, 과목 키워드를 자동 추출하는 기본 태깅 로직을 넣었습니다.
- 목록, 검색, 과목, 유형, 회차 필터, 태그 필터, 상세 보기, 수정, 삭제 흐름을 구현했습니다.

- 남은 것은 실제 사용자 계정 기준 저장과 조회를 운영 또는 스테이징에서 끝까지 한 번 더 검증하는 작업입니다.
- 운영 Auth와 허용 도메인은 현재 사용 흐름상 문제 없이 보이지만, 보관함 실사용 시나리오를 끝단까지 다시 확인하면 더 안전합니다.
