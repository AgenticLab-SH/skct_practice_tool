# 2026-07-01 일반 확장 버튼 명칭 변경

## 요청
- 일반 화면의 `테스트` 버튼 이름이 불명확하므로 `확장`으로 변경.

## 처리
- 좌측 일반 버튼 라벨을 `테스트`에서 `확장`으로 변경했다.
- 버튼 title을 `확장 안내 열기`로 변경했다.
- 보조 기능 안내 문구의 `별도 테스트 자료`, `외부 테스트 자료` 표현을 `확장 안내`, `확장 자료`로 정리했다.
- FAQ/Terms의 관련 표현도 확장 기준으로 수정했다.
- Firebase RTDB `config/siteTextConfig/utilityModal`의 운영 문구도 같은 기준으로 업데이트했다.

## 확인
- 로컬 서버에서 사이드바 라벨과 title 변경 확인.
- 운영 RTDB `utilityModal` 문구 업데이트 완료.
