# 2026-04-08 고급 모드 가이드 정리 및 OMR 통계 버튼 이동 기록
작성일시: 2026-04-08 16:55:44 KST

## 사용자 요청
- 고급 모드 페이지에 불필요한 설명이 많아 더 간단하게 정리하고 싶음
- 고급 기능 모달은 기능 활용 가이드 중심으로만 보이게 바꾸고 싶음
- `문항별 상세 통계 TXT 다운로드` 버튼은 고급 모달보다 OMR 탭 안에 두는 편이 더 쓰기 쉽다고 판단함
- 수정 후 운영에도 반영해 달라고 요청함

## 문제 판단
- 기존 고급 모달은 “현재 이용 중”, “이용권 종류”, “연장 안내”, “세부 기능 나열”이 한 화면에 섞여 있어 읽는 양이 많았음
- 실제로 사용자는 고급 모달에서 가장 먼저 알고 싶은 것이 “어디서 어떤 버튼을 누르면 되는가”인데, 이 정보가 여러 카드 사이에 흩어져 있었음
- `문항별 상세 통계 TXT 다운로드`는 복기 기능인데 고급 모달 안에 있어, 실제 작업이 이뤄지는 OMR 탭과 버튼 위치가 분리되어 있었음

## 적용 내용
- `index.html`
  - 고급 기능 모달 문구를 “고급 모드 활용 가이드” 중심으로 재작성
  - 이용권/후원 중심 문구를 제거하고, 채점 흐름과 복기 기능 위치 설명만 남김
  - `문항별 상세 통계 TXT 다운로드` 버튼과 상태 문구를 OMR 액션 영역으로 이동
  - 운영 캐시 회피를 위해 `main.js` 버전 쿼리를 갱신
- `main.js`
  - TXT 다운로드 버튼이 OMR 채점 흐름 안에서 상태 문구와 함께 동작하도록 보정
  - 고급 전용 복기 버튼들이 점수 확인 흐름과 맞춰 노출되도록 정리
- `site-text-config.js`
  - 고급 기능 모달 기본 문구를 새 가이드 구조에 맞게 갱신
- `staging/site/index.html`
  - 스테이징 사본에도 같은 구조를 반영
  - `app.bundle.js` 버전 쿼리를 갱신
- `staging/site/assets/scripts/app.bundle.js`
  - 운영본과 동일한 버튼 노출/상태 문구 로직 반영
  - `신청 안내 다시 보기` 버튼 리스너 추가

- `node --check staging/site/assets/scripts/app.bundle.js`

  - 버튼 클릭 시 상태 문구 `문항별 상세 통계 TXT 다운로드를 시작했습니다.` 노출과 실제 TXT 다운로드 발생 확인
