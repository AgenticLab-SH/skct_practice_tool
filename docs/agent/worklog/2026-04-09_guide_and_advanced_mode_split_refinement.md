# 2026-04-09 가이드와 고급 기능 분리 정리
작성일시: 2026-04-09 11:52:00 KST

## 1. 요청 요약
- 가이드 버튼 내부가 복잡하니 일반 사용 가이드와 고급 기능 설명을 분리한다.
- `고급 안내` 버튼은 `고급 기능`으로 바꾸고, 각 버튼 역할에 맞는 설명으로 다시 정리한다.
- 강조 이미지의 선과 글씨가 너무 굵어 보여서 더 가볍게 조정한다.

## 2. 이번 변경
- 사이드바 버튼 역할을 다시 정리했다.
  - `가이드`: 일반 모드 사용 흐름
  - `고급 기능`: 승인 전 사용자가 고급 모드 추가 기능과 신청/로그인 흐름을 확인
  - `고급 활용`: 승인 후 사용자가 고급 버튼 순서와 차이점을 확인
- 일반 `가이드` 모달에서는 고급 이미지와 긴 설명을 빼고, 짧은 연결 문구와 `고급 기능 보기` 버튼만 남겼다.
- 고급 기능 모달은 제목과 섹션 문구를 `일반 모드에 추가로 열리는 기능` 중심으로 다시 정리했다.
- 고급 모드 안의 활용 모달에는 강조 이미지 2장을 유지하되, 이미지 두께와 라벨 무게를 줄여 시선 부담을 낮췄다.
- OMR 하단 패널과 상태 바 버튼 문구도 `고급 활용` 방향으로 맞췄다.
