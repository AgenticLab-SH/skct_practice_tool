# 고급 모드 인라인 상태 패널 제거 및 운영 반영
작성일시: 2026-04-09 15:27:55 KST

이번 작업은 고급 모드 화면 상단에 항상 보이던 상태 패널을 제거해, 계산기와 도구 영역 높이를 다시 확보하는 것이 목적이었다.

## 문제
- 고급 모드 진입 시 타이머 아래에 `고급 모드 상태` 패널이 항상 렌더링되고 있었다.
- 이 패널이 상단 세로 공간을 차지하면서 메모장/계산기 영역이 눈에 띄게 줄어들었다.
- 사용자는 상태 바 상시 노출보다, 필요할 때 좌측 버튼으로 여는 흐름이 더 낫다고 판단했다.

## 적용 내용
- `index.html`
- 타이머 바로 아래의 `advancedModeStatusBar` 섹션 전체를 제거했다.
- 고급 모드 좌측 버튼 툴팁은 `고급 기능 열기`로 정리했다.
- `main.css`
- 제거된 상태 바 전용 스타일을 함께 삭제했다.
- `build-info.js`, `index.html`, `admin.html`, `study-archive.html`
- 자산 버전을 `202604091519`로 올려 캐시 때문에 예전 상단 패널이 남지 않게 했다.

## 검증
- `node --check main.js` 통과
- 로컬 Playwright 검증에서 아래를 확인했다.
- `advancedModeStatusBar` 없음
- `body.advanced-mode === true` 상태에서 좌측 고급 버튼 표시 유지
- 계산기 높이 `352px`, 유틸리티 높이 `333px`로 상단 공간 복구 확인

## 운영 반영 결과
- 코드 반영 커밋은 `5cbd19e`이다.
- GitHub Pages run `24175759917`가 성공으로 끝났다.
- 라이브 페이지는 `main.css?v=202604091519`, `main.js?v=202604091519`, `build-info.js?v=202604091519`를 사용한다.
- 라이브 검증에서 `advancedModeStatusBar` DOM이 사라진 것과, 고급 버튼 표시 유지, 계산기 높이 `352px`, 유틸리티 높이 `333px`를 확인했다.
