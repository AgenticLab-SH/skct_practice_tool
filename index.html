1. 실제 시험장 병행 사용 불가: 본 도구는 연습용이며 실제 온라인 SKCT 시험 창과 함께 띄워 쓸 수 없습니다.
2. 실전 감각용 모의 환경: 타이머, 메모장, 계산기, 화면 비율을 실제 흐름에 가깝게 맞춘 연습 도구입니다.
3. OMR은 채점 편의 기능: 실제 시험에는 OMR이 없고, 문제 화면에서 바로 답을 체크합니다.

🔒 고급 모드 신청 · 진입 안내

1. 승인된 경우 바로 열기
신청 이메일 또는 로그인 ID와 비밀번호를 입력하면 바로 고급 모드로 들어갑니다. 신청번호는 로그인에 쓰지 않습니다.
2. 고급에서 추가되는 것
실전 제어 확장 과목 건너뛰기, 과목 초기화, 전체 초기화처럼 반복 연습과 실전 전환에 필요한 제어가 추가됩니다.
상세 복기 정리 과목별 상세 통계, 문항별 상세 통계 TXT 다운로드, 정오표 일괄입력으로 복기 흐름을 짧게 만듭니다.
시간 감각 훈련 문항별 시간 가이드와 기록으로 어디에서 시간을 잃는지 더 세밀하게 확인할 수 있습니다.
개인 자료 보관함 자료 보관함은 고급 모드에서만 보이며, 페이지 안에서 다시 로그인한 계정만 자기 자료를 읽고 수정할 수 있습니다.
이 창의 역할
이 창은 신청과 진입을 설명합니다. 실제 버튼 사용 순서는 고급 모드 안의 고급 활용 창에서 빠르게 다시 볼 수 있습니다.
3. 처음 신청하기
후원 후 신청서를 저장하고, 승인되면 같은 창에서 바로 들어가면 됩니다.

신청 순서
1. 후원 페이지에서 금액을 확인합니다.
2. 신청서를 저장합니다.
3. 신청 이메일과 비밀번호로 상태를 확인합니다.
4. 승인 뒤 이메일 또는 로그인 ID로 고급 모드를 엽니다.
4. 신청서 저장
필수값만 입력하면 됩니다.
시작일, 닉네임, 이메일, 로그인 ID, 조회 비밀번호만 입력합니다. 로그인 ID는 일반 ID나 이메일 형식 모두 사용할 수 있습니다.
같은 비밀번호를 신청 조회와 고급 로그인에 함께 씁니다. 신청번호를 기억할 필요는 없습니다.
5. 신청 상태 확인
신청 조회는 신청 이메일과 비밀번호로만 합니다. 승인 후 고급 로그인은 이메일 또는 로그인 ID로 진행합니다.
문의가 있으면 zhdlsqpdj@gmail.com로 보내주세요.

✨ 고급 모드 빠른 사용

이 창은 신청 절차가 아니라, 들어온 뒤 어디를 누르는지 다시 찾는 용도입니다.
고급 모드에서는 상단 상태 바, OMR 아래 복기 버튼, 자료 보관함, 우측 실제환경 여백이 추가됩니다.

자료 보관함 위치
자료 보관함은 OMR 아래가 아니라 더보기 메뉴에서 따로 엽니다. 고급 모드가 열린 브라우저에서만 보입니다.
1. 결과부터 확인
채점 및 통계 확인으로 맞은 수, 정답률, 건너뜀, 못 푼 문제를 먼저 봅니다.
2. 과목별 약점 확인
과목별 상세 통계로 어떤 영역이 흔들렸는지 바로 확인합니다.
3. TXT로 기록 남기기
문항별 상세 통계 TXT 다운로드로 복기 기록을 저장합니다.
4. 반복 연습 준비
정오표 일괄입력, 과↺, 전↺, 시간 가이드를 조합해 같은 세트를 다시 돌립니다.

☕ 우리의 공동 툴,
쾌적하게 지켜나가기

우리가 함께 지켜나가는 '결제/광고 없는' SKCT 연습 공간입니다! ✍️

저 역시 여러분과 같은 취준생이기에, 연습할 때 방해받지 않고 온전히 집중할 수 있도록 상업적 배너 광고나 결제 유도를 일절 넣지 않고 사비로 100% 무료 서버를 가동 중입니다.

최근 동시 접속자가 늘어나면서 서버 유지의 부담도 조금씩 커지고 있지만, 끝까지 이 '광고 없는 무료 개방' 원칙을 고수하려 합니다.

만약 이 쾌적한 '모두의 툴'이 여러분의 합격 준비에 약간이나마 유용하셨다면, 투네이션(또는 메일)을 통해 '커피 한 잔 ☕'의 후원에 동참해 주시길 조심스레 부탁드립니다.

이 공간이 다음 달에도 거뜬히 유지되려면 여러분 한 분 한 분의 십시일반 참여가 필수적입니다! 💕
